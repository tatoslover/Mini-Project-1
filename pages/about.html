Advanced Statistics Explained

PER (Player Efficiency Rating): A per-minute rating that summarizes a player's statistical accomplishments. League average is 15.0.
VORP (Value Over Replacement Player): How much a player contributes compared to a replacement-level player. Higher is better.

True Shooting % (TS%): Measures shooting efficiency accounting for 2-pointers, 3-pointers, and free throws. League average is ~57%.
Usage %: Percentage of team plays used by a player while on the court. Higher usage indicates more offensive involvement.

Win Shares: Estimate of wins contributed by a player. Combines offensive and defensive contributions.
Box Plus/Minus: Per-100-possession estimate of points contributed above league average. 0 is average, +5 is All-Star level.

Presentation Q&A

Requirements & Design Process
Started with user stories for NBA fans, created wireframes, and followed an iterative development approach. The design focused on mobile-first responsive layout with clear information hierarchy.
Data Sources
Player statistics from NBA Stats API stored in JSON format. Data includes advanced metrics like PER, VORP, True Shooting %, and traditional stats.
Data Processing
  • JavaScript array methods (filter, map, sort, reduce) process raw data.
  • Chart.js visualizes aggregated statistics.
  • Real-time filtering updates DOM elements dynamically.
User Interactions
Tabbed navigation, searchable dropdowns, real-time filtering, sortable data tables, interactive charts, and responsive comparison tools.
HTML Elements Used

CSS Features Used
CSS Grid, Flexbox, animations (@keyframes), custom properties (variables), gradients, transforms, transitions, responsive design (media queries), and custom scrollbars.
